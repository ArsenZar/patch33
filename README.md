# patch33

Адаптивна браузерна 3D-презентація шеврона.

## GitHub Pages

Сайт не потребує збірки або Jekyll. Для публікації:

1. Відкрийте `Settings` → `Pages`.
2. У `Build and deployment` виберіть `Deploy from a branch`.
3. Виберіть гілку `main` і папку `/(root)`.

Файл `.nojekyll` змушує GitHub Pages публікувати статичні файли напряму.
